About This Project

AI-Powered Music Discovery

This project combines cutting-edge machine learning with real-world data to create a unique music recommendation experience. Using TF-IDF (Term Frequency-Inverse Document Frequency) and cosine similarity algorithms, we analyze song lyrics, genres, and musical features to find songs that match your taste.

Our standout feature is weather-based recommendations - the system detects your location, analyzes current weather conditions, and suggests music that perfectly complements the mood of the moment.

Technologies Used:

Python Flask Scikit-learn TF-IDF OpenWeatherMap API HTML5/CSS3 JavaScript
